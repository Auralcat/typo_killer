# Typo Killer
### _qu'est que ce_

#### Usage

With elixir installed, open `typo_killer` folder and use the following commands on your terminal:
```
mix deps.get
mix compile
iex -S mix
```

Then just: `TypoKiller.find_typos("path/to/folder")`
